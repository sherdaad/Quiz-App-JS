# Quiz-App-JS
quiz application using HTML,CSS &amp; Javascript

HTML & JavaScript Quiz Website

This is a simple quiz website built using HTML and JavaScript.

It allows users to answer multiple-choice questions.

The quiz shows instant results after submission.

JavaScript is used for quiz logic and score calculation.

HTML is used for structure and layout.

The design is simple and user-friendly.

No external libraries are used.

Suitable for beginners to understand basic concepts.

Easy to run in any modern web browser.

Created for learning and practice purposes.
